# House-Price-Prediction
House Price Prediction using Machine Learning Algorithm(Random Forest, Linear Regression


## Domain
Real Estate

## Technologies Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## Description
This project predicts house prices using the Ames Housing dataset.
It includes data preprocessing, visualization, model training,
and evaluation.

## How to Run
1. Install required libraries.
2. Open the notebook in Jupyter Notebook or Google Colab.
3. Run all cells.
